### LIKE with wrong wildcard

```abap
SELECT ... WHERE xyz LIKE 'PATTERN*'.
```
instead of
```abap
SELECT ... WHERE xyz LIKE 'PATTERN%'.
```

### NP or CP with wrong wildcard

```abap
IF xyz CP 'PATTERN%'.
```
instead of
```abap
IF xyz CP 'PATTERN*'.
```
