### EC CI_SUBRC can be removed

If the SUBRC is checked, the pseudo comment 'EC CI_SUBRC' can be removed.

Example:

```abap
SELECT * FROM TADIR. "#EC CI_SUBRC  <- can be removed, since SUBRC is checked afterwards
IF sy-subrc = 0.
  ..
ENDIF.
```
