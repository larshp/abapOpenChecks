### CLEAR as first use of variable

Checks if a variable is cleared in it's first usage after it has been defined.

Note: Only works for local variables in global classes.

Example:

```abap
..
METHOD foobar.

  DATA lv_index TYPE i.
  CLEAR lv_index. 
  ..

ENDMETHOD.
..
```
